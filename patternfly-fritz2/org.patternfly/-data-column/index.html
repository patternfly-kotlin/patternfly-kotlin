DataColumn

class DataColumn<T>(columns: Columns<T>, id: String?, baseClass: String?, text: String) : Column<T> , WithIdProvider<T>
Content copied to clipboard

Column to show the actual data of an item in the ItemStore. Use this column any number of times to render the properties of the items.

The column uses display functions for the header and cells. One of the data columns should use the ItemStore.idProvider to assign an element ID. This ID is referenced by various ARIA labelledby attributes.

If you want to make the column sortable, use one of the sortInfo() function to specify a Comparator.
