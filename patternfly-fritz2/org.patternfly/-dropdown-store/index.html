DropdownStore

class DropdownStore<T> : RootStore<List<Entry<T>>>
Content copied to clipboard

Store containing the data shown in a dropdown. The data is wrapped inside instances of Entry. An entry is either an Item or a Group of Items. An Item can have additional properties such as an icon, a description or a disabled state.
